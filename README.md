# AI Image Captioning Project using Transformers
This project is an AI-powered Image Captioning system that automatically generates captions for images using Deep Learning and Transformer models.
The model understands the content of an image and produces human-like textual descriptions.
## Features
- Upload any image
- Automatically generates captions
- Uses Vision Transformer (ViT) + GPT-2
- Deep Learning based project
- Runs easily on Google Colab
- Beginner-friendly AI project
##Technologies Used
- Python
- Transformers
- PyTorch
- Vision Transformer (ViT)
- GPT-2
- Google Colab
- PIL (Python Imaging Library)
## How It Works
1. User uploads an image
2. Image is processed using Vision Transformer
3. Features are extracted from the image
4. GPT-2 generates a meaningful caption
5. Caption is displayed as output
